# IT Asset Management System 🖥️🔧

Професійна ERP-система для IT-відділу, побудована на Django. Дозволяє вести облік майна, керувати кадрами, обробляти сервісні заявки та автоматизувати роботу через Telegram & ШІ.

## 🚀 Основні модулі

### 📋 Inventory (Майно)
* Облік техніки, моніторів та серверного обладнання.
* Автоматичне закріплення за працівниками.
* Експорт даних у Excel та зручний пошук.

### 👥 Staff (Кадри)
* Електронний штатний розпис.
* Зберігання сканів наказів про призначення/звільнення.
* **Розумна чистка**: автоматичне видалення старих файлів з диска при оновленні наказів.
* Керування сертифікатами КЕП та історія кар'єрних переміщень.

### 🛠️ Service (Сервіс та ремонти)
* Журнал заявок на ремонт та заправку картриджів.
* Формування актів передачі техніки в сервісні центри.
* AJAX-інтерфейс для швидкого отримання/видачі обладнання.

### 🤖 Telegram Bot + Gemini AI
* Інтегрований бот для прийому заявок.
* Використання **Gemini 1.5 Flash** для розпізнавання вільного тексту користувача ("У нас в 2610 принтер гуде") та автоматичного створення структурованої заявки в БД.

### 📂 Docs & Resources
* Централізоване зберігання інструкцій та посилань на Google Drive.
* Directory: довідник внутрішніх телефонів та адрес.

## ⚙️ Технічні особливості
* **Оптимізація**: Використання `select_related` та `prefetch_related` для високої швидкості роботи.
* **Безпека**: Валідація РНОКПП, номерів телефонів та захищені вебхуки.
* **Моніторинг**: Контроль використання дискового простору (ліміт 512 МБ) безпосередньо в інтерфейсі.

## 🛠️ Встановлення та запуск
1. Встановіть залежності: `pip install -r requirements.txt`
2. Налаштуйте ключі в `settings.py`: `TELEGRAM_BOT_TOKEN`, `GEMINI_API_KEY`.
3. Виконайте міграції: `python manage.py migrate`
4. Зберіть статику: `python manage.py collectstatic`
